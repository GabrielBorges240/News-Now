# News Now

Aplicação que exibe notícias em tempo real usando a NewsAPI.

Demo:  https://gabrielborges240.github.io/News-Now/

## Funcionalidades
- Listagem de notícias atualizadas
- Filtro por categoria
- Tratamento de erros da API

## Tecnologias
- HTML5
- CSS3
- JavaScript
- NewsAPI

## Como executar
1. Insira sua chave da API no código
2. Abra o arquivo `index.html` no navegador
